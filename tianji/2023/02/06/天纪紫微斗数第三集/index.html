天纪紫微斗第三集笔记

作者 1628466962@qq.com

第6-1集笔记​

判断出生时辰​

  • 发旋 子午卯酉 偏左或者居中 寅申已亥 偏右 辰戌丑未 两个旋或两个以上 都是从后面看
  • 十二宫与方位

紫微星在丑宫​

  • 紫微星在丑宫

  • 如果命宫廉贞落陷 贪狼落陷 或宫命对宫有 廉贪落陷 半空折翅格 大限大约在30左右

  • 半空折翅格 1.先天带的疾病 2.自杀

  • 天相 为天上的宰相 非常好的辅佐人才 不想权位

    • 天相坐命 位高无权
    • 面相 瘦高 瘦瘦高高 高挑形
    • 大运 流年 有天相 表示该运处于位高无权的现象 老是干副的

  • 命主迁移宫

    • 紫微星 左辅右弼 坐迁移宫说明在外地发展比本地好很多 如果坐夫妻宫说明对方很好

  • 命主夫妻宫

    • 廉贞 贪狼 坐夫妻宫 夫妻问题很大

  • 廉贞 七杀 半路死亡 半路埋尸

  • 廉贞 破军 水中作冢 在水中作坟墓

  • 廉贞 贪狼 横夭 偏向于先天的疾病 2 自杀

  • 杀破狼煞星再遇廉贞 更煞

  • 煞星遇庙旺 说明是煞星很亮 看得到的煞 凶中藏吉

  • 煞星落陷 说明是煞星很暗 看不得到的煞 吉中藏凶 在劫难逃

  • 命主子女宫

    • 太阴化忌 白天出月亮 对宫 田宅宫 太阳陷 日月反背 无子女

  • 日月反背

    • 在命宫
      • 工作黑白颠倒 披星戴月 晚上比白天还忙
      • 六亲无靠
      • 性情非常刚燥
    • 在父母宫(后面讲)
      • 父母不在身边
      • 父母双亡
        • 先亡父 先亡母看印堂
        • 看眉毛 左边父 右边母

  • 命主十年(23-32 33-42)

    • 未来十年无大运 大限在前十年
    • 33-42 日月反背 化忌 无子 无大限 大限在 22-32
    • 22-32 廉贞落陷 半空折翅
    • 连续两个十年煞
    • 煞都在夫妻与子女宫
    • 有自杀倾向

  • 命主疾厄宫

    • 陀罗落陷 铃星 很煞
    • 坐寅宫 主肺 左脚
    • 太阴星辰宫化忌 胃一定不好
    • 廉贪坐巳宫 脾也不好

  • 忧能伤肺 思能伤脾

  • 阳宅

    • 厨房在西北 死别 死夫

    • 厨房在西南角 死本人

    • 客厅在西北角 生离 丈夫不在 如军人 警察 船员 飞行员

    • 客厅在东边

      • 1.生不出儿子

      • 2.儿子不喜欢在家 象客人

    • 如果子女在正东方位 阳宅东边是厨房 主长子夭折

    • 如果丈夫 子女宫化忌 无儿子 如果有儿子 另一半命中有儿子 且像妈妈 如果像爸爸 房子东边又是厨房 主夭折

      • 化解1 阳宅化解
      • 化解2 医学化解

  • 命主福德宫

    • 武曲七杀
      • 男孩子 福德宫 表示个性刚强不靠祖业
      • 女孩子 福德宫 一辈子靠自已

  • 武杀会 武曲七杀 命宫

    • 女子 主孤单
    • 离祖发迹
    • 男 武职大利
    • 商人 劳多功少

  • 命主 十年大运(33-42)

    • 太阴化忌 无儿子
    • 父母凶主丧亡
    • 姻婚凶 太阴代表太太化忌 婚姻一定完蛋
    • 天喜 解神 喜神化忌
    • 所以命主 婚姻不是生离就死别

  • 婚姻不好解法

    • 六为阴数之极
    • 九为阳数之极
    • 婚姻凶解法
      • 找二婚的结婚
      • 长7岁以上

  • 合八字

    • 合八字与八字合是两回事
    • 八字分为合 与不合
    • 先天二婚的要找一个不合的八字
    • 合命方法以案例命主为例
      • 命主父母宫 天同天梁 找一个 命宫 天同天梁的女孩子 为母子合
      • 命主兄弟宫 巨门文昌 找一个命宫 巨门文昌的女孩为兄弟合
      • 命主朋友宫 天机 天魁 找一个命宫 天机 天魁的女孩子为朋友合

  • 命中无问题不要改大运顺其自然

紫微星在寅宫​

  • 紫微星在寅宫

  • 命主命宫

    • 紫微天府
    • 不管宫位 紫微天府 为紫府坐垣 命宫如果在对面 七杀 叫七杀朝斗
    • 紫府坐垣 与七杀朝斗 都是大贵的
    • 紫府坐垣 位列三台 宰相 坐命为君
    • 七杀朝斗 将星入命 威震边疆

  • 紫府坐垣

    • 在官䘵宫 很好
    • 在财帛宫 这两颗星是财库不是财得 权很大 只是权势很大没有多大的财

  • 命主子女宫

    • 太阳陷 晚上出太阳

  • 太阳落陷

    • 在田宅宫 有火象
    • 在父母宫 父有亡之象
    • 在夫妻宫
      • 女眼角纺两根 先生很好 多离别 聚少离多
      • 面大骨粗声莎 先先亡之象
    • 子女宫
      • 有儿子 离别
      • 化忌死
    • 财帛宫 不能做意
    • 象批 眼镜不好

  • 武官入庙 最好大利

  • 武曲 贪狼入庙不是煞星

  • 命主田宅

    • 巨门星 口舌 官司 化忌 一定找官司 逢天马 车祸

  • 命主十年大运(33-43)

    • 太阳落陷看不见 对宫巨门 化忌 太阳表示先生不在了 聚少离多 化忌 离婚
